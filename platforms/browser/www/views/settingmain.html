About

"Collegecounsel - college list builder" is an app that assist with the college search process by providing relevant college information as well as features that enable the users to create their personal college list through inputing several parameters.


Most of the school information are obtained from the college official websites. However, some school's info are obtained from a dataset from Kaggle.com

App Usage

  • "Explore" Page: click on the college name to view more info about that college. Use the search bar to type schools you want to look up Use the dropdown next ot the search bar to rank schools

  • "Build My List" Page: GPA, test score (you don't need to specify which one, just input the numerical value), and number of results to display are required. If you want to use the optional parameters, check the box in front of the one you want to use. After the list is rendered, click on the college name to view more info and use the add button to add a school to your list which is displayed on the third page

  • "College List" Page: first use "Build List" page to add schools to your list. Then you can view them on this page. Click on the college name for more info. Click on the red "x" to delete a school
